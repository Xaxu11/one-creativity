## Project Description
OneCreativity is a multi-page demo platform that connects clients with studios and freelancers.
Built using only HTML, CSS, and JavaScript (no frameworks, no libraries).

## Pages
- Home (index.html)
- Services
- Studios
- Freelancers
- Booking
- About
- Contact

## Features
- Responsive layout (Grid/Flex)
- Motion backgrounds (CSS animation)
- Scroll reveal (IntersectionObserver)
- Modal previews (services/studios/freelancers)
- Favorites (LocalStorage)
- Booking summary saved (LocalStorage)
- Recently viewed (LocalStorage)
- Theme toggle Dark/Light (LocalStorage)
- Skeleton loading shimmer
- Form validation (JS)
- Back to top button

## How to Run
Open index.html in a browser or use VS Code Live Server.
