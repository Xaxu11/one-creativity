About

Modern UI + strong fundamentals

OneCreativity is a demo multi-page platform that connects clients to studios and freelancers. This final project focuses on clean UI/UX and core web development skills.

Platform Idea

A single place to browse creative providers, compare services, save favorites, and submit booking requests.

What I implemented

Multi-page navigation, responsive layout, animated backgrounds, scroll reveal, DOM rendering, filters, modal previews, LocalStorage favorites & booking summary, and validation.

Skills Demonstrated

  • Semantic HTML structure (header/nav/main/section/footer)
  • Responsive CSS layout (Flexbox + Grid)
  • JavaScript DOM manipulation + event handling
  • LocalStorage (favorites + booking summary + recent views)
  • Modal UI interactions + accessibility basics
  • Scroll reveal (IntersectionObserver)
  • Theme toggle (Dark/Light)
  • Skeleton loading shimmer
